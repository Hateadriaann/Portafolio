Temas Clave del Curso

1. Modelo Relacional y Normalizacion

Aprender a estructurar las tablas para evitar redundancia. Reglas de 1FN, 2FN y 3FN para bases de datos eficientes.

2. Lenguaje SQL (DDL y DML)

El estándar mundial. Crear tablas (CREATE) y manipular datos (SELECT, INSERT, UPDATE, DELETE).

3. Programacion en BD (Stored Procedures)

Crear lógica programable directamente en el motor de base de datos usando Triggers y Procedimientos Almacenados.
