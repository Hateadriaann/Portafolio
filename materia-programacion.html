PROGRAMACIÓN WEB

El arte de construir internet. Aprender a estructurar, diseñar y dar vida a sitios y aplicaciones web dinámicas.

Aplicación en el Mundo Real

Hoy en día, todo vive en la web. Esta materia te prepara para los roles con mayor demanda laboral inmediata Frontend Developer y Web Master.

Frontend

Crear interfaces atractivas y responsivas (que se vean bien en celular y PC) usando HTML, CSS y JavaScript.

Backend

Entender cómo un formulario de registro envía tus datos a un servidor y cómo recibirlos para guardarlos en una base de datos.

Temas Clave del Curso

1. Maquetación Web (HTML5, CSS3)

La estructura semántica de una pagina (DOM) y el diseño visual (Flexbox, Grid, Media Queries) para crear sitios modernos.

2. Scripting del Lado del Cliente (JavaScript)

Darle vida a la página: validación de formularios, animaciones y manipulación de elementos en tiempo real sin recargar la pagina web.

3. Programación del Lado del Servidor

Introducción a tecnologías como JSP, PHP o Node.js para recibir peticiones (GET/POST) y conectar la web con una base de datos.
